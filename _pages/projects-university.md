## First Year
### 2D Platform Adventure Game

This is a solo 2D project that took 2-3 months to develop. The game features basic movement mechanics, animated pathfinding enemies that do damage, various audio features, and a personal follower, acting as a light source for the player. Thoughtful unique mechanics and atmospheric lighting effects earned approval from examiners.

### 3D Linear Horror Game

This is a solo 3D project that took 2-3 months to develop. The criteria for this assignment was to simply show a comprehensive understanding of ProBuilder tools and general level creation. The game world is made using ProBuilder tools, featuring intricate lighting components to create a horror atmosphere that stands out. The game features intense enemy > player pursuit mechanics, audio features, and displays my skill in level design and rewarding player choices.
